# ticalcode / Kulono Aesthetic Design System

## ticalcode (soul.your-domain.com)

- **Brand**: ticalcode — Where AI inherits wisdom, not just memory
- **Background**: Pure black #000000
- **Color Palette**:
  - Background: #000000 (pure black)
  - Secondary Background: #0a0a0a
  - Tertiary Background: #111111
  - Border: rgba(255,255,255,0.06)
  - Primary Text: #e8e8e8
  - Secondary Text: #888888
  - Muted Text: #555555
  - Accent: #c8c8c8
  - Accent Hover: #ffffff
- **Fonts**: Inter (modern sans-serif) + JetBrains Mono (monospace)
- **Core Principles**: Zero emoji, zero purple, zero gradients, Text is the visual
- **Layout**: Fixed navbar 56px, max-width 960px, 24px padding
- **Interaction**: Frosted glass navbar (backdrop-filter: blur(20px))
- **Buttons**: White-on-black (primary) / Transparent black border (secondary) / Full transparent (ghost)
- **Selection**: White semi-transparent rgba(255,255,255,0.12)
- **Hover**: Color transition 0.15s ease

## Kulono (kulono.com)

- **Brand**: Kulono — Where extraction meets return (Ku → Lono cycle philosophy)
- **Background**: Dark gray #0a0a0a
- **Color Palette**:
  - Left glow: rgba(180, 50, 20, 0.12) — Red (Ku)
  - Right glow: rgba(20, 140, 80, 0.10) — Teal (Lono)
  - Axis gradient: Transparent → Red → Gold → Teal → Transparent
  - Text: #e0e0e0
- **Fonts**: Inter (light 200/300)
- **Core Elements**: Breathing animation glow (8s cycle), center axis divider (4s pulse), large headlines clamp(3rem-7rem), letter-spacing 0.35em, uppercase
- **Mood**: Ritual/sacred, bipolar power balance, cycle philosophy

## Common Points

- Dark theme (pure black / near-black)
- Inter font family
- Minimalism, zero decorative redundancy
- Text itself is a visual element
- Fine micro-interactions (0.15s transition)

## Differences

- ticalcode: More "product", structured, informational, with navigation/cards/button systems
- Kulono: More "art", immersive, breathing glow, ritualistic, single-page concept showcase
